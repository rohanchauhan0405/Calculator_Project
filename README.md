# Calculator_Project

## PROJECT SUMMARRY:--
1. Performed basic arithmetic operations including Addition, Subtraction, Multiplication, Division, and Percentage calculations.
2. Implemented dynamic user input handling through JavaScript Event Listeners.
3. Created an interactive calculator interface with responsive button controls.
4. Used DOM Manipulation to update and display real-time calculations.
5. Implemented Clear (C) functionality to reset calculations.
6. Applied JavaScript's expression evaluation to process mathematical operations.
7. Designed a clean and user-friendly interface using CSS styling and Flexbox layout.
